# Minimal blog starter

Ez egy ultra-letisztult statikus blog váz:
- Bejegyzések (lista + két minta poszt)
- Fotók (gyűjtemények, rács)
- Listák (kattintható, <details>/<summary>)
- Info oldal

## Futtatás
Nyisd meg a `index.html`-t böngészőben.

## Publikálás
- GitHub Pages / Netlify / Vercel: statikus site-ként kirakható.

## Testreszabás
- `style.css`-ben: háttér, betű, zöld accent
- Menüpontok a layoutban vannak minden oldalon.
